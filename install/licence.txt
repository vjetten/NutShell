Nutshell version 3.0 and higher can handle the new PCRaster install (beta 3 onwards)
This installer also installs MapEdit 1.11 (compile date 270209)

********************************************
WARNING:
The config file nutshell.ini in the folder: [your profile name]\\application data\\nutshell backed up as nutshell.### where ### is the previous version number.
A new file is created on the first startup. An icon will be placed on your desktop.
Note: if you cannot install NutShell in the PCRatser directory, any other directory will also work!
